+ Habilidades:

En HTML aprendi que son las etiquetas semanticas, la estructura basica de un proyecto HTML a realizar la maquetacion de una idea en codigo, a identificar el SEO que nos ayudara a posicionar nuestra pagina, a enlazar nuestras HTML a un index e interactuar con ellos, el entender el codigo mas crudo de una pagina web.

En CSS3 fue un proceso creciente de aprendizaje ya que iniciamos a darle estilos sencillos a nuestros HTML utilizando solamente atributos, cambiandole de color, hasta que utilizamos la primera tecnologia que fue Box Modeling, la cual nos ayudaria a empezar a posicionar los elementos dentro de nuestro proyecto, despues vimos la genealidad de Grid y Flexbox la cual nos facilitaria aun mas el interactuar con nuestros elementos y empezar a analizar el lado responsive y como ayudaban estas nuevas tecnologias.

En Git fue un cambio en como podiamos trabajar y compartir un proyecto, en el cual iniciamos utilizando GIT para crear commits de nuestros avances del proyecto e identificarlos, despues crear nuestras cuentas en GitHub en el cual pudimos interactuar ya de una manera mas profesional el trabajo colaborativo de nuestros proyectos y en nuestro caso la revision de los mismos .

con Boostrap vimos como funcionaba una libreria y como se utilizaba en proyectos mas avanzados, ya que boostrap nos ayudaba con toda la parte responsive de nuestro sitio web y asi mismo a utilizar sus moldes disponibles para ahorrarnos tiempo codeando, investigando su documentacion y como modificar todo lo que nos ofrece esta libreria.

con SASS practicamos una nueva manera de integrar estilos de una manera mas organizada y practica a nuestros proyectos ya que SASS es un procesador de CSS lo que significa que nos ayudaria a hacer el codigo en CSS dandole nosotros primero los parametros que queremos tener, asi mismo ver su funcionalidad en cuando a poder reutilziar codigo dentro de SASS.
